## Help Centre

<Update label="9 May 2026" description="New URL: eggz.ai/help (was help.eggz.ai)">
  ### What changed

  - The Help Centre now lives at **`eggz.ai/help`** instead of **`help.eggz.ai`**. Bookmarks and search results pointing at the old subdomain are **301-redirected** path-for-path to the new URL — for example, `help.eggz.ai/unscrambled-tabs/install` → `eggz.ai/help/unscrambled-tabs/install`. No content was lost.
  - Internal links across the Help Centre, the home page, and every Update entry below were rewritten to the new `/help/...` paths.

  ### Why

  - **SEO consolidation:** moving from a subdomain to a subdirectory keeps all link equity and topical authority on the eggz.ai apex domain instead of splitting it across `eggz.ai` and `help.eggz.ai`.
  - **One brand, one host:** the Dashboard, marketing, and the Help Centre all share the eggz.ai origin.

  ### How it works

  - **Hosting:** Mintlify serves the docs; a Cloudflare Worker on the eggz.ai zone proxies `/help/*` (and the required Mintlify well-known and asset paths) to `eggz.mintlify.app/help/*`.
  - **Redirect:** A second Cloudflare Worker on `help.eggz.ai/*` issues HTTP **301** to the matching `eggz.ai/help/*` URL (path, query, and fragment preserved).
  - **Extension:** the Unscrambled Tabs extension's **Performance & RAM → Learn more** link now opens `eggz.ai/help/unscrambled-tabs/performance-and-ram#performance-modes-explained`.
</Update>

## Unscrambled Tabs (Chrome extension)

<Update label="v1.1.3" description="9 May 2026 · Help Centre deep link, icon pipeline, About version">
  ### Release

  - **Chrome Web Store / manifest** version set to **1.1.3** (matches the built package users install).

  ### UX

  - **Performance & RAM → Learn more** opens this Help Centre at the [**Performance modes**](/help/unscrambled-tabs/performance-and-ram#performance-modes-explained) section (Manual, Balanced, Aggressive scenarios). The extension uses a configurable help base URL (default **`https://help.eggz.ai`**).

  ### About

  - **Settings → About → Version** uses the live **Manifest V3** version from Chrome (`chrome.runtime.getManifest().version`) so it always matches the store and the browser. Packaged builds also embed the same semver from the extension manifest at compile time so UI and tests stay aligned.

  ### Branding / build

  - **Toolbar and Web Store icons** are generated from a single **master PNG** via **Sharp**, letterboxed on black with `contain` fit for non-square sources. **`pnpm run build`** runs icon generation first; the step fails clearly if the source asset is missing.

  ### Testing

  - Automated tests cover extension version resolution when the Chrome manifest API is stubbed and when running outside the extension runtime.
</Update>

<Update label="v1.1.2" description="RAM-first workspace performance">
  ### Performance

  - **Performance mode:** **Manual** (default), **Balanced**, or **Aggressive** — controls when eligible inactive tabs are discarded via Chrome [`tabs.discard`](https://developer.chrome.com/docs/extensions/reference/api/tabs).
  - **Free RAM now:** one-shot discard of **safe** inactive tabs from Settings — tabs stay on the strip; Chrome reloads when you activate them.
  - **Multi-window inactive sweep:** scheduled work considers tabs across **normal windows**, not only the active window.

  ### Windows & workspace binding

  - Each Chrome window can keep a **workspace binding** so the side panel and dashboard show tabs for the workspace tied to **that** window when you run several workspaces at once.

  ### Safety & privacy

  - **Never** auto-targeted for suspension helpers: active, pinned, audible, already discarded, `chrome://` / extension pages, non-http(s) URLs, and domains you list under **Protected domains**.

  ### Permissions

  - **`windows`** added so the UI can resolve the current Chrome window reliably for routing and bindings ([`chrome.windows.getCurrent`](https://developer.chrome.com/docs/extensions/reference/api/windows)).

  ### Documentation & testing

  - Internal manual UAT checklist for RAM / performance QA (aggregate metrics only — no logging of sensitive tab data).
  - Expanded automated tests for **discard eligibility** rules.
</Update>

<Update label="v1.1.1" description="Verified CRX uploads & signing">
  ### Security

  - Opted in to Chrome Web Store **Verified CRX Uploads** for tamper-evident publishing.
  - Releases are signed with the project’s **RSA 2048-bit** key before upload; **unsigned packages are rejected**.

  ### Developer experience

  - **`pnpm run pack:crx`** builds and signs the extension in one step (requires local signing key per release runbook).
  - **Chrome Web Store upload guide** documents a repeatable release checklist for maintainers.
</Update>

<Update label="v1.1.0" description="Custom emoji, side panel performance, themes, protected domains">
  ### Custom favicon emoji

  - Choose a **custom emoji** in Settings for tabs with missing favicons (replaces the generic globe) across workspaces, sidebar, and dashboard.

  ### Performance (side panel)

  - **No white flash** when switching workspaces; large workspace switches stay smooth.
  - **Background tab updates** (URL, title, favicon) are **batched** instead of saving on every event.
  - **Remembers** scroll position, active view, and expanded workspaces between opens.

  ### Dark & light mode

  - **Theme overhaul** using shared design tokens; **dark mode** aims for **WCAG AA** contrast.
  - **Light mode:** toasts, tooltips, and modals match the theme; export/import password flows inherit the active theme.

  ### Protected domains

  - **Protected domains** list in Settings — keep important sites (email, calendar, etc.) from being auto-suspended.

  ### Stability

  - Safer behaviour when **no browser window** is available during tab operations.
  - Fewer startup errors when **restoring session** tabs.
</Update>

<Update label="v1.0.0" description="Initial public release">
  - Workspace manager with tab grouping and organisation.
  - AI-powered tab search and assistant.
  - Session time travel with auto-snapshots every **5 minutes**.
  - Tab deduplication with instant switch banner.

  - **Encrypted backup and restore** (AES-256-GCM).
  - **Workona** import support.
  - Keyboard shortcuts (e.g. **⌘K** search, **⌘J** AI assistant).
  - Dashboard with resizable panels for tabs and resources.
